## Description

Stunning design, connected to the blockchain, metamask pairing, interaction with smart contracts, sending Ethereum through the blockchain network, writing solidity code, and much more.

#### Technologies

- React
- Tailwind
- Solidity
- Hardhat
- Metamask
- Visual studio code
